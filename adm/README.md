# H2 Admin

Admin panel for managing users and generated PDFs.

## Run

1. Ensure API is running at `http://localhost:3000`
2. Install and run:

```bash
cd adm
npm install
npm run dev
```

3. Open http://localhost:4000

## Features

- **Users**: View user list (deviceId, PDF count, created at), delete user
- **PDFs**: View PDF list (filename, size, pages, deviceId), delete PDF

No login required.
